# Test Checklist

- Confirm Excel is 32-bit: File > Account > About Excel.
- Close all Excel processes before installation.
- Install using a normal Windows user account.
- Open Excel and confirm the Sarwar Smart PCMS tab.
- Test WBS coloring on a copied workbook.
- Test XER import using a non-confidential sample.
- Compare imported row counts with Primavera.
- Test dates, calendars and activity IDs.
- Test uninstall and confirm the ribbon disappears.
- Do not distribute externally until all XER export results are verified by re-importing into a test P6 database.
